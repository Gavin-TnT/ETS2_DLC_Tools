# ETS2 DLC Tools

基于PyQt6的欧洲卡车模拟2（ETS2）DLC管理工具。

## 界面布局 - 简化版本

### 仅导航栏模式
应用程序现在只保留了左侧功能导航栏，移除了所有其他界面元素，专注于核心导航功能。

#### 导航按钮
- **✓ 勾选图标** - "已安装DLC"按钮
  - 功能：显示已安装的DLC列表
  - 状态：默认选中状态
  - 样式：选中时显示蓝色背景

- **✖ 叉号图标** - "未安装DLC"按钮  
  - 功能：显示未安装的DLC列表
  - 状态：可切换选中
  - 样式：选中时显示蓝色背景

- **⚙ 齿轮图标** - "设置"按钮
  - 功能：打开设置选项
  - 状态：可切换选中
  - 样式：选中时显示蓝色背景

#### 导航栏特性
- **竖直排列**：三个按钮在左侧顶部竖直排列
- **单选模式**：同一时间只能选中一个功能按钮
- **悬停效果**：鼠标悬停时显示灰色背景
- **点击反馈**：点击时显示深灰色背景
- **圆角设计**：按钮具有5px圆角
- **工具提示**：鼠标悬停显示功能说明
- **紧凑设计**：更小的按钮尺寸（35x35px）

### 极简布局结构
```
┌─────────────────────────────────────┐
│ 左侧导航菜单 (800x600px 窗口)        │
│                                     │
│  ┌─────────────────────────────────┐ │
│  │ ✓ 已安装DLC                    │ │
│  │ ✖ 未安装DLC                    │ │
│  │ ⚙ 设置                         │ │
│  └─────────────────────────────────┘ │
│                                     │
│  （仅保留三个导航按钮，移除列表显示）   │
└─────────────────────────────────────┘
```

## 技术特性

- **框架**：PyQt6
- **布局**：QHBoxLayout分栏布局（20%左侧菜单 + 80%右侧内容）
- **样式**：最小化CSS样式表
- **响应式设计**：动态调整左侧菜单宽度，保持20%比例
- **动态内容切换**：堆栈式页面管理，支持菜单驱动的内容更新
- **阴影效果**：QGraphicsDropShadowEffect专业级阴影
  - 模糊半径：20px
  - 阴影颜色：半透明黑色（RGBA: 0,0,0,80）
  - 偏移：仅右侧4px
- **图标**：Unicode字符图标（✓ ✖ ⚙）
- **组件**：QToolButton实现导航按钮
- **窗口大小**：1000x700px（响应式布局）

## 使用说明

1. 启动应用程序后，只显示左侧导航菜单
2. 点击三个功能按钮切换不同的DLC显示模式
3. 控制台会输出相应的操作日志
4. 窗口可以自由调整大小，但保持窄窗口设计
5. 点击关闭按钮直接退出应用程序
